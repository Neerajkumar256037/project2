🛒 E-Commerce Website Project

This is a simple front-end E-commerce website built using HTML and CSS. It includes 3 main pages: Home (E-commerce), Login, and Registration.

📄 Pages Included

🏠 1. E-commerce (Home Page)

Displays website logo and navigation menu
Sections: Home, Shop, Blog, Contact
Shows products or fashion-related content
🔐 2. Login Page

User login form
Fields:
Username / Email
Password
Button for login
📝 3. Registration Page

New user registration form
Fields:
Name
Email
Password
Confirm Password
Submit button to create account
🛠️ Technologies Used

HTML5
CSS3
📁 Project Structure

project-folder/ │── index.html (E-commerce Home Page) │── login.html (Login Page) │── register.html (Registration Page) │── style.css (CSS Styling)

🚀 How to Run

Download or clone this repository
Open "index.html" in your browser
Navigate between pages using menu links

Simple and clean design
